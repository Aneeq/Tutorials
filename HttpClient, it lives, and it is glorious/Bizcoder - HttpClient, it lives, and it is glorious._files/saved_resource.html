                                                                      Darrel Miller Mod Terrence • 6 years ago

                                                                      Here's my almost fool proof way to resolving these types of issues.
                                                                      Step 1: Identify what media type you are using. - application/json
                                                                      Step 2 : Identify what Formatter Web API is using for that media type.
                                                                      Step 3: Identify what object serializer the Formatter is using.
                                                                      Step 4: Test serializing and deserializing your List<person> using that serializer.

                                                                      99% of the time these issues are not a Web API issue but some stupid serializer problem. Which is why I don't use object serializers for my wire formats :-)
