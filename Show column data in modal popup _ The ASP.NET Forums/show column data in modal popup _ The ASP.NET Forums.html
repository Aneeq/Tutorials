    Hi Hozefa Unwala，

    hozefa_unwala

    I have 6 columns in MVC datatable. i can show 5 columns in datatable but would like to show 6th column(Notes) in modal popup. Can someone help me how to show that data in modal popup. below is the example

    According to your description, I suggest you could use Partial View to get more data from sql db to view.

    Then you could use model popup to show the Partial View when user click the details button.

    I make a demo, you could refer to it.

    Index View:

    @model IEnumerable<WebApplication1.Models.Employee>
@{
    ViewBag.Title = "Index";
}



<div id='myModal' class='modal'>
    <div class="modal-dialog">
        <div class="modal-content">
            <div id='myModalContent'></div>
        </div>
    </div>

</div>
<table class="table">
    <th>FirstName</th>
    <th>LastName</th>
    <th>Salary</th>
    <th>Department</th>
    <th>Position</th>
    <th>ViewDetails</th>
    <th>Details</th>
    @foreach (var item in Model)
    {

        <tr>
            <td>
                @Html.DisplayFor(modelItem => item.FirstName)
            </td>
            <td>
                @Html.DisplayFor(modelItem => item.LastName)
            </td>
            <td>
                @Html.DisplayFor(modelItem => item.Salary)
            </td>
            <td>
                @Html.DisplayFor(modelItem => item.Department)
            </td>
            <td>
                @Html.DisplayFor(modelItem => item.Position)
            </td>
            <td>
                @Html.DisplayFor(modelItem => item.ViewDetails)
            </td>
            <td>
           
                <a href="javascript:void(0);" class="anchorDetail" data-id="@item.Id">Details</a>

            </td>
        </tr>
    }

</table>


@section scripts
{
    <script src="~/Scripts/jquery-1.10.2.min.js"></script>
    <script src="~/Scripts/bootstrap.js"></script>
    <script src="~/Scripts/bootstrap.min.js"></script>
    <script>
        var TeamDetailPostBackURL = '/Authors/myModal';
        $(function () {
            $(".anchorDetail").click(function () {
                var $buttonClicked = $(this);
                var id = $buttonClicked.attr('data-id');
                $.ajax({
                    type: "GET",
                    url: TeamDetailPostBackURL,
                    contentType: "application/json; charset=utf-8",
                    data: { "Id": id },
                    datatype: "json",
                    success: function (data) {
                        $('#myModalContent').html(data);
                        $('#myModal').modal('show');

                    },
                    error: function () {
                        alert("Dynamic content load failed.");
                    }
                });
            });
        });

    </script>

}

    Controller:

      public ActionResult Index()
        {
            ViewBag.type = 0;
            return View(db.Employees.ToList());
        }

        public ActionResult myModal(int Id)
        {
            ViewBag.type = 1;
            Employee employee = db.Employees.Find(Id);
            if (employee == null)
            {
                return HttpNotFound();
            }
            return PartialView("_myModal", employee);

        }

    _myModal Partial View:

    @model WebApplication1.Models.Employee

<div>
    <div class="modal-header">
        <button type="button" class="close" data-dismiss="modal" aria-label="Close"><span aria-hidden="true">&times;</span></button>
        <h4 class="modal-title">Details</h4>
    </div>
    <div class="modal-body">
        <table class="tablemodel">
        
                <tr>
                     <tr>
                         Notes:
                     </tr>
                    <tr>
                        @Html.DisplayFor(Model => Model.Notes)
                    </tr>
          
        </table>
    </div>
    <div class="modal-footer">
        <button type="button" class="btn btn-default" data-dismiss="modal">Close</button>
    </div>
</div>
